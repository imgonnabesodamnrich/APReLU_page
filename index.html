[Paper Notes] Deep Residual Networks with APReLU for Mechanical Fault Diagnosis under Complex Conditions

An Adaptive Deep Learning Framework for Mechanical Fault Diagnosis

Paper Title: Deep Residual Networks With Adaptively Parametric Rectifier Linear Units for Fault Diagnosis

Journal: IEEE Transactions on Industrial Electronics (TIE), Vol. 68, No. 3, 2021

DOI: 10.1109/TIE.2020.2972458

1. Introduction and Background

In modern industrial systems, condition monitoring of rotating machinery (e.g., gearboxes, bearings) is crucial for ensuring production safety. With the development of deep learning, data-driven fault diagnosis methods are gradually replacing manual feature extraction. However, mechanical equipment in actual industrial environments often faces complex variable working conditions, such as frequent speed fluctuations and load switching. These factors cause the statistical properties of vibration signals to be highly non-stationary. This paper proposes a deep residual network model incorporating Adaptively Parametric Rectifier Linear Units (APReLU) to improve the feature discrimination accuracy under complex variable conditions.

2. Feature Learning Challenges under Variable Conditions

Under variable working conditions, mechanical fault diagnosis faces two core challenges:

  • Intra-class Dissimilarity: The vibration signal distributions of the same fault type may vary greatly in the time or frequency domain under different speeds or loads.
  • Inter-class Similarity: Different types of faults (e.g., gear wear vs. bearing pitting) may exhibit highly similar vibration characteristics under specific conditions.

Traditional deep learning models (like ResNet, CNN) use fixed non-linear activation functions (like ReLU), applying a uniform transformation to all input signals. This "one-size-fits-all" approach struggles to effectively separate highly confused signals in the feature space, limiting the flexibility of the model's decision boundaries.

3. Core Innovation: APReLU Activation Function

3.1 Motivation of APReLU

To enhance the model's adaptability to different input signals, the Adaptively Parametric Rectifier Linear Unit (APReLU) was designed. Unlike traditional activation functions, the negative slope of APReLU is neither a preset constant nor a globally shared parameter, but a dynamically generated variable based on the current input sample.

3.2 Sub-network Structure and Parameter Adaptation

APReLU integrates a lightweight sub-network to achieve parameter adaptation. The specific process is as follows:

  1. Global Information Extraction: The input feature map first goes through Global Average Pooling (GAP) to extract global spatial information of each channel.
  2. Dynamic Slope Prediction: The sub-network (consisting of Fully Connected layers and Sigmoid mapping) calculates a specific slope coefficient $\alpha$ for each input channel based on the extracted information.
  3. Non-linear Transformation: The model applies a non-linear mapping to the current sample according to the dynamically generated slope, achieving a "customized" feature transformation effect.

4. ResNet-APReLU Architecture

The model uses a Deep Residual Network (ResNet) as the backbone. The identity connection mechanism of ResNet effectively solves the vanishing gradient problem in deep neural networks. The paper embeds the APReLU activation function into each residual block (ResBlock). This integration allows the network to not only possess deep semantic abstraction capabilities but also perform dynamic gain adjustment and feature calibration for samples under variable conditions during each layer's feature transmission.

5. Experimental Analysis and Verification

5.1 Planetary Gearbox Experimental Design

The study used a planetary gearbox dynamic simulation test rig for data acquisition. The experiment set up 8 different health states (including normal, tooth wear, root crack, etc.) tested under 9 combinations of 3 speeds and 3 loads. Additionally, additive Gaussian white noise of different intensities was added to the original signals to evaluate the model's robustness.

5.2 Comparative Experimental Results

ResNet-APReLU was compared with traditional ConvNet and ResNets using activation functions like Sigmoid, Tanh, ReLU, LReLU, and PReLU:

  • Accuracy Improvement: Under various Signal-to-Noise Ratios (5dB, 3dB, 1dB), ResNet-APReLU maintained the highest testing accuracy.
  • Generalization Performance: Compared to fixed-slope models, the adaptive model exhibited stronger robustness when facing unseen working conditions.

5.3 Feature Visualization Analysis

The t-SNE algorithm was used to perform dimensionality reduction and visualization on the last layer's features. The results showed that ResNet-APReLU significantly widened the feature distance between different faults and compressed the feature distribution space of the same faults under different conditions, forming much more compact and clear clustering results.

6. Conclusion

By introducing an adaptive activation mechanism into the residual structure, the proposed ResNet-APReLU model effectively solves the difficulty of feature discrimination of vibration signals under complex variable working conditions. Its excellent performance in planetary gearbox fault diagnosis reflects the advantage of dynamically adjusting non-linear transformation parameters for non-stationary industrial signals, providing a highly flexible neural network construction approach for intelligent industrial data analysis.
